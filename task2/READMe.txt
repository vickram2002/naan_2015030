Introduction
This project is a web-based Personal Information Form. Its purpose is to collect basic personal details from users and display the information entered. It serves as a simple example of an HTML form with validation and JavaScript for dynamic content display.

Prerequisites
Before you begin, ensure you have the following:

A modern web browser.
Features
Input Fields: The form collects the following personal information:

Name
Email
Phone Number
Address
Validation: Basic input validation is implemented using HTML5 attributes (e.g., required).

Data Display: After the user submits the form, JavaScript dynamically displays the entered information below the form.
